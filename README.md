# Prosjekt-oppgave

# Oppgave beskrivelse 
I dette prosjektet skal jeg utvikle et digitalt sosialt chatteforum der brukere kan opprette konto, logge inn og kommunisere med andre. Plattformen skal fungere som et sted hvor venner og nye bekjentskap kan sende meldinger, dele tanker og holde kontakt.

# hva brukeren skal kunne gjøre:
Registrere seg og logge inn med brukernavn og passord
Opprette en personlig profil
Søke etter og legge til venner
Sende og motta meldinger i sanntid
Delta i samtaler i både private og offentlige chatrom

# Målet med prosjektet
Målet med nettsiden er å vise at jeg kan bruke det jeg har lært i IT til å lage noe ekte og funksjonelt. Jeg vil vise at jeg kan lage en nettside der folk kan lage konto, logge inn og chatte med venner på en enkel og trygg måte. Prosjektet skal vise at jeg forstår hvordan man bygger en nettside fra bunn av, både teknisk og med et brukervennlig design.
